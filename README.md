# Gatsby projects

## Creating an Agency Site

### The Setup

1. Install [Gatsby][github-gatsby] globally

    ```shell
    npm install -g gatsby-cli
    ```
    - 
       

2. Create new Gatsby site

    ```shell
    gatsby new agencyDemo https://github.com/gatsbyjs/gatsby-starter-hello-world
    ```
3. run
    
    ```shell
    gatsby develop
   ```
